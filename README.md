# 📦 Supply Chain Analytics – Demand Forecasting & Anomaly Detection

## 📖 Project Overview

This project is developed as part of the Infotact Technical Internship Program.

The objective is to analyze historical supply chain data, detect operational anomalies, and forecast future demand using time series analysis and machine learning techniques.

---

## 🎯 Objectives

- Clean and preprocess supply chain data
- Perform exploratory data analysis (EDA)
- Detect anomalies using statistical techniques
- Forecast future demand using time-series models
- Build an interactive Streamlit dashboard
- Generate business insights for inventory optimization
